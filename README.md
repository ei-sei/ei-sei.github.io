# Cloud Engineer Portfolio

Hi, I'm Sheikh Khaled Ahmed - a DevOps and Cloud Engineer passionate about building and automating infrastructure. This portfolio is designed to showcase my projects, skills, and hands-on experience across cloud platforms, containerisation, networking, and security.

The site itself is built with React and Vite, following an MVC architecture with a dark-themed UI, responsive design, and project filtering.


## Tech Stack

- **Frontend:** React, React Router, CSS Modules
- **Build:** Vite
- **Deployment:** GitHub Pages
- **Architecture:** MVC (Models, Views, Controllers)

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173/`

```bash
npm run build      # Production build
npm run preview    # Preview production build
npm run deploy     # Deploy to GitHub Pages
```

## 🏗️ MVC Architecture

```
src/
├── models/
│   └── projectsData.js          # Project data and state models
├── views/
│   ├── components/              # Reusable UI components
│   │   ├── Navbar.jsx
│   │   ├── Hero.jsx
│   │   ├── ProjectCard.jsx
│   │   ├── ProjectDetail.jsx
│   │   ├── Skills.jsx
│   │   ├── Contact.jsx
│   │   └── Footer.jsx
│   └── pages/                   # Page components
│       ├── HomePage.jsx
│       ├── ProjectsPage.jsx
│       └── ProjectDetailPage.jsx
├── controllers/
│   └── projectController.js     # Business logic and data operations
├── styles/
│   └── global.css              # Global styles and theme variables
└── App.jsx                      # Main app with routing
```
